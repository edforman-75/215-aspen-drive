Critical Path

Decisions, Sequence, and What Cannot Wait

Start here: The deck structural repair is the trigger for everything on the exterior. It must happen first, and it forces the stucco decision immediately — you will need a stucco contractor on site regardless. Every other exterior decision flows from or alongside this work.

Exterior Critical Path

PHASE E1 — REQUIRED FIRST

Deck Structural Repair

Non-negotiable starting point. Structural integrity before all cosmetic work.

  • Engage structural contractor to assess and repair deck framing
  • This work disturbs adjacent stucco — patching is required on completion
  • Triggers Phase E2 immediately

Reversibility: N/A — maintenance, not elective

Can defer: No

↓
PHASE E2 — REQUIRED, TRIGGERED BY E1

Stucco Repair — and Tier Decision

The deck repair requires stucco patching. A stucco contractor is on site. This is the moment to decide tier:

  • GOOD: Patch damaged area only, repaint to match existing texture
  • BETTER: Patch + skim coat entire exterior to smooth finish + repaint
  • BEST: Patch + skim coat entire exterior + repaint (same as BETTER at this step)
A fresh patch against weathered original stucco will be visible. Doing the full surface while the contractor is mobilized is the logical and economical step. Marginal additional cost over patch-only.

Reversibility: Skim coat to smooth is permanent — cannot restore original texture. Repaint is reversible.

Can defer full stucco: Yes, but patch is not deferrable and mobilization makes full surface the practical choice.

↓ (in parallel with E2)
PHASE E3 — TESTS BEFORE COMMITMENT

Three Tests That Must Happen Before Major Decisions

Run these tests while structural/stucco work is underway. Results inform the Phase E4 decisions.

Test 1: Block Paint Strip — One Wall

  • Strip paint from most visible wall section (minimum 4'×4', ideally full wall)
  • Use BioBlast chemical strip (Alan Barr, Wood River Valley)
  • Evaluate: color, uniformity, texture, light at different times of day
  • Cost: ~$150-$300 | Required before committing to full strip
  • If result is poor: repaint to match, stay with BETTER painted block
  • If result is good: proceed to full BioBlast treatment + clear sealer

Test 2: Slider Cladding Paint — One Slider

  • Clean, prime with aluminum bonding primer, paint one slider dark bronze/black
  • Cost: ~$100-$150 | Observe for 1-2 months before committing to all four
  • If paint holds: do remaining three sliders (~$400-$800 total)
  • If paint fails or aesthetics aren't sufficient: replace sliders with Milgard A250
↓
PHASE E4 — MAJOR DECISIONS (informed by tests)

Commit to Tier and Scope

With test results in hand, commit to the overall exterior tier and each element decision:

  • Block: Natural concrete (BEST) or painted dark (BETTER)?
  • Sliders: Painted cladding (bridge) or new Milgard A250 (BETTER/BEST)?
  • Windows: Keep existing (painted frames, GOOD/BETTER) or Milgard A250 or Kolbe VistaLuxe AL (BEST)?
  • Railings: Restain balusters (GOOD) or cable rail retrofit (BETTER/BEST)?
  • Entry doors: Restain (GOOD) or new Simpson glass panel (BEST)?

These decisions can be mixed — you are not locked to one tier across all elements.

↓
PHASE E5 — CONSTRUCTION

Execute Based on Decisions

  • Full stucco skim coat (if BETTER/BEST) or repaint texture (if GOOD)
  • Block stripping — full BioBlast treatment (if BEST test was positive)
  • Slider replacement — Milgard A250 installed (if selected)
  • Window replacement — Kolbe VistaLuxe AL installed (if selected)
  • Cable rail retrofit — Feeney CableRail (if selected)
  • Entry door replacement (if selected)
  • Deck restain — top surface, underside (spray), beam ends
  • Paint: stucco, block (if painted), railings, door

Windows and sliders can be scheduled independently of stucco/block — different contractors, different timing.

Where to Start Right Now

1. Engage structural contractor for deck assessment
Everything exterior flows from this. Get the scope and timeline before making any other exterior decisions.
2. Run the block paint strip test simultaneously
Contact Alan Barr (Wood River Valley) to strip one wall section. This test result is required before you can commit to BEST tier. It costs ~$200 and could save $1,000+ if the result is disappointing.
3. Paint one slider cladding
Low-cost, reversible, immediate visual feedback. Tests the dark-frame palette for $100-$150 before committing to $10,000-$16,000 in slider replacement.
4. Order Room & Board swatches now (free)
Swatches take 1-2 weeks to arrive. Order everything — Jasper sofa, Quinn chair, Afton leather, Hudson walnut — so they're on hand when decisions are ready. No commitment required.
5. Search for vintage furniture while structural work happens
Set alerts on 1stDibs for Drexel Declaration, Paul McCobb Planner Group. Contact Sweet Modern about current inventory. Good vintage pieces move quickly — sourcing can start immediately and run in parallel with everything else.
6. Get a window quote before committing to tier
Contact Glass Masters (118 Lewis St, Ketchum, (208) 726-1420) for a Kolbe VistaLuxe AL quote on the 6 upstairs windows. The actual installed price will sharpen the BEST tier decision considerably.
